Professional Data Modelling

  • Real-World Evolution - handle multiple fact tables (Sales + Returns)

  • Relationship Best Practices - one-to-many, filter direction, troubleshooting

Essential DAX Formulas

  • Measures vs Columns - know when to use which (critical!)
  • CALCULATE Function - the most powerful DAX tool for dynamic filtering
  • Time Intelligence - YTD, Last Year, Growth % and trend analysis
  • Business Calculations - % of Total, Rankings, Running Totals
  • Error Handling - common mistakes and how to fix them
